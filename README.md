


# TripLive
SIAT FINAL PROJECT / TRIPLIVE / SPRINGBOOT

> 당신을 위한 맞춤형 현지 정보
> 내일의 여행을 준비하는 우리에게, 여행은 TripLive

​                                                 

## 개요

> 여행자들에게 안전정보와 같은 현지 정보를 전달하는 사이트 입니다.
